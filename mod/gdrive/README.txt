= Google Drive =
Combines different additions into one plugin

== Features ==
- Google Drive Widget show contents of specified directory
- Extends Elgg search with documents in Google Drive (in  progress)

== Configuration ==
1. Get a Google Account
2. Setup Google Drive API keys (Server Apps API key, Browser Apps API key, Client ID and Client Secret)
3. Add a Google Drive widget and specify Folder ID
